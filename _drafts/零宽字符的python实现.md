# 零宽字符的 python 实现

## 原理

### 零宽字符

零宽字符是一种在浏览器中不打印的字符，大致相当于 display: none ,在许多文本应用中也不显示，比如邮箱、QQ、微信、文本编辑器等

这里有三种零宽字符 -- 零宽空格、零宽连字、零宽不连字

```
零宽字符在浏览器中对应的转义字符

零宽空格    --- &#8203;  
零宽不连字  --- &#8204;
零宽连字    --- &#8205;
```
